# Day 22.2：前端流水线串联（解析→确认→自动管线）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将后端 7 步管线（推导公式→质量过滤→校验→异常检测→摘要→AI分析→图表推荐）串联到前端，用户确认结构后自动推进，无需手动逐步调用 API。同时在上传和 review 之间补齐解析步骤。

**Architecture:** 管线由 `usePipeline` hook 管理状态机，`PipelinePanel` 负责渲染进度。`TaskReviewPage` 确认结构后通过 `onConfirmed` 回调触发管线。`App.tsx` 在管线完成后提供"查看结果"跳转。

**Tech Stack:** React, TypeScript, Fetch API

- [ ] **Step 3: 实现 run 函数**

核心逻辑：
1. 重置所有步骤为 pending
2. 顺序遍历 steps，每步 set status="running"
3. 特殊处理 `rules` 步骤：GET 获取规则列表，若 `has_gap` 则自动 POST acknowledge-gap，并将 `validate` 步骤标记为 skipped
4. 其他步骤：POST/GET 对应 endpoint
5. 每步成功 → done + 提取消息；失败 → failed + 错误信息，立即终止
6. 全部完成 → allDone=true

```typescript
const run = useCallback(async (taskId: string) => {
  setRunning(true);
  setAllDone(false);
  const updated = [...STEPS.map(s => ({ ...s, status: "pending" as StepStatus, message: "" }))];
  setSteps([...updated]);

  let hasGap = false;
  for (let i = 0; i < updated.length; i++) {
    const step = updated[i];
    if (step.key === "validate" && hasGap) {
      step.status = "skipped";
      step.message = "无可用公式规则，已跳过";
      setSteps([...updated]);
      continue;
    }
    step.status = "running";
    step.message = "";
    setSteps([...updated]);

    try {
      if (step.key === "rules") {
        const res = await fetch(`${API_BASE}/tasks/${taskId}${step.endpoint}`);
        if (!res.ok) throw new Error("获取规则失败");
        const data = await res.json();
        if (data.has_gap) {
          hasGap = true;
          await fetch(`${API_BASE}/tasks/${taskId}/formula-rules/acknowledge-gap`, {
            method: "POST",
            headers: { "Content-Type": "application/json" },
            body: JSON.stringify({ acknowledged: true }),
          });
          step.message = `通过 ${data.passed ?? 0} 条，过滤 ${data.filtered ?? 0} 条（空规则集已确认跳过）`;
        } else {
          step.message = `通过 ${data.passed ?? 0} 条规则`;
        }
      } else {
        const res = await fetch(`${API_BASE}/tasks/${taskId}${step.endpoint}`, {
          method: step.method,
          headers: step.method === "POST" ? { "Content-Type": "application/json" } : undefined,
          body: step.method === "POST" ? "{}" : undefined,
        });
        if (!res.ok) {
          const err = await res.json().catch(() => ({}));
          throw new Error((err as { detail?: string }).detail ?? "请求失败");
        }
        const data = await res.json();
        step.message = data.total_issues !== undefined
          ? `发现 ${data.total_issues} 个问题`
          : data.total !== undefined
            ? `生成 ${data.total} 个图表`
            : data.accepted_rules !== undefined
              ? `接受 ${data.accepted_rules} 条规则`
              : "完成";
      }
      step.status = "done";
    } catch (e) {
      step.status = "failed";
      step.message = e instanceof Error ? e.message : "执行失败";
      setSteps([...updated]);
      setRunning(false);
      return;
    }
    setSteps([...updated]);
  }
  setRunning(false);
  setAllDone(true);
}, []);
```
