### 1.2 边界场景

| 场景 | 操作 | 预期 |
|------|------|------|
| 空文件上传 | 上传非 .xlsx 文件 | 弹窗"仅支持 .xlsx 文件" |
| 后端未启动 | 关闭后端，使用前端 | 显示"无法连接服务器"而不是白屏 |
| 上传中断 | 上传中途断网 | 显示"上传失败"+重试按钮 |
| 管线某步失败 | Mock 模式下模拟失败 | 该步显示 ✗，后续步骤标记 skipped |
| 刷新页面 | 在结构确认页刷新 | 回到上传页（当前无状态持久化） |
| 快速双击 | 快速点击"确认结构"2 次 | 不会重复执行管线 |
| 超长文本 | 单元格内容 1000+ 字符 | 网格不撑破，内容截断+省略号 |
